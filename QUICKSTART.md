# 빠른 시작 가이드

## 5분 안에 시작하기

### 1단계: 프로그램 실행 (30초)

**방법 A: Python으로 실행**
```bash
python capcut_motion.py
```

**방법 B: EXE 파일 사용 (추천)**
```bash
# 먼저 EXE 빌드
build.bat

# 그 다음 실행
dist\CapCut_모션_자동화.exe
```

### 2단계: 파일 선택 (1분)

1. CapCut **완전히 종료** (중요!)
2. CapCut 프로젝트 폴더로 이동:
   ```
   C:\Users\[사용자명]\AppData\Local\CapCut\User Data\Projects\[프로젝트명]\
   ```
3. `draft_content.json` 파일 찾기
4. 프로그램에서 "파일 선택" 클릭 → 파일 선택

### 3단계: 설정 선택 (1분)

**초보자 권장 설정:**
- 모션 패턴: "기본 줌인"
- 시작 스케일: 1.04
- 끝 스케일 최소: 1.08
- 끝 스케일 최대: 1.10
- 이동 범위: 0.05
- ✓ 원본 파일 백업 생성 (체크)

**"모션 적용하기" 클릭!**

### 4단계: 파일 교체 (2분)

프로그램이 생성한 파일들:
- `draft_content_motion.json` (새 파일)
- `backups/draft_backup_20240101_120000.json` (백업)

**파일 교체 방법:**

```bash
# 1. 원본 파일 이름 변경 (추가 백업)
draft_content.json → draft_content_original.json

# 2. 새 파일 이름 변경
draft_content_motion.json → draft_content.json
```

### 5단계: CapCut에서 확인 (1분)

1. CapCut 실행
2. 프로젝트 열기
3. 타임라인에서 클립 확인
4. 재생하여 모션 효과 확인

---

## 트러블슈팅

### ❌ "파일이 선택되지 않았습니다"
→ 파일 선택 버튼을 먼저 클릭하세요

### ❌ "모션이 적용되지 않아요"
→ CapCut이 완전히 종료되었는지 확인하세요

### ❌ "백업 폴더가 없어요"
→ 자동 생성됩니다. 프로젝트 폴더 내 `backups/` 확인

### ❌ "효과가 너무 강해요/약해요"
→ 스케일 값 조정:
- 약하게: 시작 1.02, 끝 1.04~1.06
- 강하게: 시작 1.05, 끝 1.10~1.12
